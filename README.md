# Registration Page (Frontend Project)

A responsive registration page built using HTML, CSS, and JavaScript.  
This project demonstrates basic form layout, client-side validation setup, and clean UI design suitable for beginner to intermediate frontend development practice.

## Live Demo link:
https://alishaikh54.github.io/registration-page-ui/

## Features
- Responsive layout for mobile and desktop
- Clean and simple UI design
- Custom form validation (HTML5 validation disabled using `novalidate`)
- Basic input field styling and error handling
- Beginner-friendly project structure

## Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

## Project Structure
<pre>
  .
├── index.html
├── style.css
└── script.js
</pre>


## Getting Started (Run Locally)
1. Clone the repository:
  https://github.com/AliShaikh54/registration-page-ui.git
2. Open `index.html` in your browser.

## Use Case
This project can be used as a base template for:
- User registration forms
- Login/Signup UI
- Form validation practice
- Frontend portfolio projects
